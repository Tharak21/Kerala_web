# 🌴 Kerala Tourism Website

A responsive tourism website showcasing the beauty, culture, and destinations of Kerala, built using HTML, CSS, and JavaScript.

This project was developed as part of the **23CSE113 – User Interface Design** course.

## 🚀 Live Preview

Add your deployed link here:

https://github.com/Tharak21/Kerala_web

## 📸 Features

- Responsive modern design
- Smooth scrolling navigation
- Animated UI elements
- Destination showcase sections
- Mobile-friendly layout
- Interactive hover effects
- Clean and organized structure

## 🛠 Built With

- HTML5
- CSS3
- JavaScript

## 🖥 Installation

Clone the repository:

```bash
git clone https://github.com/Tharak21/Kerala_web.git
```

Move into the project folder:

```bash
cd Kerala_web
```

Open `index.html` in your browser.
